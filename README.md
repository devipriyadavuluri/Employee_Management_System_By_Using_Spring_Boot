# Spring Boot Employee Management System

This is a Spring Boot based Employee Management application.

Features:
- Add new employees
- View employee details
- Update employee information
- Delete employees

Technologies Used:
- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL / H2 Database

Project Structure:
- Controller layer
- Service layer
- Repository layer
- Entity layer

Purpose:
This project was developed to understand Spring Boot fundamentals,
REST APIs, and layered architecture.
